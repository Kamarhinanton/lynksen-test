This is a "Lynksen-test" project. Created by Next.js.

<h2 id="requirements">Requirements</h2>

node version: 20.9.0

npm version: 10.1.0

<h2 id="installation">Installation</h2>

```bash
npm install
# or
yarn install
```

<h2 id="getting-started">Getting Started</h2>

For develop:

```bash
npm run dev
# or
yarn dev
```

Prepare for production:
```bash
npm run build
# or
yarn build
```

```bash
npm run start
# or
yarn start
```
